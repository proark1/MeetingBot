How it works

Three steps to meeting intelligence

From API call to actionable insights in under a minute.

1

Send the bot

Call the API with your meeting URL. MeetingBot joins Zoom, Google Meet, or Teams within seconds using a real headless browser.

2

Record & transcribe

The bot silently records audio and video, transcribes every speaker in real-time using Gemini's speech recognition model.

3

Receive insights

Claude AI analyzes the transcript and delivers a structured summary, action items, and custom analysis via webhook or API response.

Developer-first

Simple API.
Powerful results.

One POST request sends a bot into any meeting. Our REST API is designed to be intuitive, with SDKs for Python and JavaScript to get you started in minutes.

Get your API key → View full docs
example.py
import meetingbot

# Initialize client with your API key
client = meetingbot.Client("sk_live_...")

# Send a bot into any meeting
bot = client.bots.create(
  meeting_url="https://meet.google.com/abc-defg-hij",
  bot_name="Notetaker",
  webhook_url="https://yourapp.com/webhook"
)

# Bot joins, records, and delivers results
print(bot.id)  # bot_01jk...

# Retrieve transcript & AI summary later
result = client.bots.get(bot.id)
print(result.summary)  # AI-generated insights
