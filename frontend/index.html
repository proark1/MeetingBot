MeetingBot API

Drop a bot into any Zoom, Google Meet, or Teams call. Get back a full transcript and AI analysis. No database required — you store the data.

Google Meet Zoom Microsoft Teams Claude / Gemini Multi-Tenant Stripe + USDC
Interactive API Docs ReDoc Reference Register Account

How it works

1

Send the URL

POST your meeting URL and optional webhook_url to /api/v1/bot.

2

Bot joins

A headless Chromium browser joins the call, records audio, and monitors captions.

3

Transcribe & analyse

Audio is transcribed and analysed by Claude or Gemini AI.

4

Get results

Full payload delivered to your webhook_url, or poll GET /api/v1/bot/{id}.

Bot lifecycle

joining → in_call → call_ended → done or error

Results are kept in memory for 24 hours — save them to your own storage before then.

Quick start

Create a bot

curl -X POST https://your-meetingbot.com/api/v1/bot \ -H "Content-Type: application/json" \ -d '{ "meeting_url": "https://meet.google.com/abc-defg-hij", "bot_name": "Notetaker", "webhook_url": "https://your-app.com/webhook/meeting-done", "template": "default" }'

Poll for status

curl https://your-meetingbot.com/api/v1/bot/{id}

Webhook payload (POST to your webhook_url when done)

{ "event": "bot.done", "data": { "bot_id": "550e8400-...", "status": "done", "participants": ["Alice", "Bob"], "transcript": [{ "speaker": "Alice", "text": "...", "timestamp": 2.0 }], "analysis": { "summary": "...", "action_items": [{ "task": "...", "assignee": "Alice" }], "decisions": [...], "sentiment": "positive" }, "ai_usage": { "total_tokens": 4200, "total_cost_usd": 0.021 } } }

API Endpoints

POST/api/v1/botCreate a bot & join a meeting
GET/api/v1/botList all in-memory bots
GET/api/v1/bot/{id}Full details — transcript, analysis, stats
DELETE/api/v1/bot/{id}Stop & remove a bot
GET/api/v1/bot/{id}/transcriptRaw transcript only
GET/api/v1/bot/{id}/recordingDownload WAV audio
POST/api/v1/bot/{id}/analyzeRe-run AI analysis
POST/api/v1/bot/{id}/askQ&A on the transcript
POST/api/v1/bot/{id}/followup-emailDraft a follow-up email
GET/api/v1/bot/{id}/export/markdownExport as Markdown
GET/api/v1/bot/{id}/export/pdfExport as PDF
POST/api/v1/webhookRegister global webhook
GET/api/v1/templatesList analysis templates
GET/api/healthHealth check

Templates

Pass template in your bot creation request. Use prompt_override for a custom prompt.

default · sales · standup · 1on1 · retro · kickoff allhands · postmortem · interview · design-review

Authentication & Accounts

Register an account to receive an sk_live_... API key. Use it as a Bearer token on all requests.

POST/api/v1/auth/registerCreate account → get API key
POST/api/v1/auth/loginEmail + password → JWT for web UI
GET/api/v1/auth/meAccount info + balance
POST/api/v1/auth/keysGenerate new API key
Authorization: Bearer sk_live_<your-api-key>

Billing & Credits

Pre-fund your credit balance via Stripe card or USDC (ERC-20). Credits are deducted per bot run.

GET/api/v1/billing/balanceCurrent balance + transaction history
POST/api/v1/billing/stripe/checkoutCreate Stripe checkout session
GET/api/v1/billing/usdc/addressGet USDC deposit address (1 USDC = $1)

Admin (restricted)

Platform administration — only admin accounts can access these endpoints.

GET/api/v1/admin/walletView platform USDC collection wallet
PUT/api/v1/admin/walletSet/update platform USDC wallet
GET/api/v1/admin/configList platform configuration

Web admin panel: /admin
